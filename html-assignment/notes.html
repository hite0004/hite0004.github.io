HTML Tags with Description

List Tag(li)
This tag is used to create list of our content.
Ordered List tag(ol)
This tag is used to list our content in a perticular order.
Unordered List tag(ul)
This tag is used to list our content in no perticular order that is by showing bullet points right before our content.
Image tag(img)
This tag is used to add an image into our HTML document.
Title tag(title)
This tag is used to add a title of our webpage.
